# 맛집 예약 서비스
## 필수 기능
- [ ] 사용자는 특정 맛집을 예약할 수 있다.
- [ ] 모든 사용자는 예약 현황을 확인할 수 있다.
- [ ] 사용자 본인은 자신이 한 예약의 상세 정보까지 확인할 수 있다.
- [ ] 사용자는 본인의 예약만 수정/삭제할 수 있다.

## 프로그래밍 요구사항
- [ ] h2를 이용하여 데이터를 관리한다.
- [ ] 테스트 코드를 통해 애플리케이션이 의도대로 동작하는지 검증한다.
- [ ] 공휴일 정보 조회 외부 API를 연동한다.
